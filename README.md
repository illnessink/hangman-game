
# HangMan

This is a web application that uses the Wordnik api to generate a random word to play a game of hangman with.  

## Technologies Used  


- HTML5
- CSS3
- JavaScript
- JQuery
- Google Fonts
- [Wordnik API](https://developer.wordnik.com/docs#!/words/getRandomWord)
